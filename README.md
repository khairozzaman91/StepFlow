# 🏦 NexusBank - Modern Fintech Dashboard

A professional banking dashboard built with **React**, **Tailwind CSS**, and **Lucide Icons**. 

### ✨ Features:
- **Dynamic Balance:** Real-time balance updates.
- **Transfer Money:** Fully functional modal to send money.
- **Transaction History:** Add, view, and delete transactions.
- **Persistence:** Data stays saved even after refreshing (LocalStorage).
- **Responsive Tabs:** Switch between Dashboard, Transactions, and Cards.

### 🛠️ Tech Stack:
- React.js (Vite)
- Tailwind CSS
- LocalStorage API
